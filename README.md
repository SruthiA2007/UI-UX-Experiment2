# UI-UX-Experiment2

## Aim:
To design a basic mobile and web layout using Gestalt principles, UI elements, usability principles, and a custom brand logo.

## Algorithm:

1.Select an existing case study.

2.Create a brand logo and choose colors/fonts.

3.Apply Gestalt principles in the layout.

4.Design mobile and web screens.

5.Add UI elements and design patterns.

6.Define interaction behaviors.

7.Check usability and consistency.

8.Export the final design.

## Output:
[UI & UX epx-2.pdf](https://github.com/user-attachments/files/28542308/UI.UX.epx-2.pdf)

## Result:

A user-friendly mobile and web layout was successfully designed with a brand logo, UI elements, Gestalt principles, and usability features to enhance the user experience.
